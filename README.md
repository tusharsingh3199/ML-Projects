# ML PROJECTS LIST:
# 1. Malaria Cell Detection using TensorFlow

This project focuses on detecting malaria-infected cells from microscopic images using **deep learning techniques**. Two approaches were implemented:
1. **Custom CNN Base Model** – built from scratch using TensorFlow/Keras. Results: F1 Score = 96%, Accuracy = 96%.
2. **Transfer Learning with Fine-Tuning** – leveraging pre-trained models (ResNet50) to improve accuracy and reduce training time. Results: F1 Score = 97%, Accuracy = 97%.

# 2. Brain Tumor Segmentation using 3D Unet and Swin UNETR

This project focuses on segmentation of different brain tumor using BraTS 2019 dataset. The 2 model implemented:
1. **3D UNet** – Dice Score = 0.8
2. **Swin UNETR** – Dice Score = 0.7
